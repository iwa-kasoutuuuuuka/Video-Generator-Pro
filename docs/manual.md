# 取扱説明書 (User Manual) - Shorts Video Generator Pro

## 1. 導入手順
### 1.1 前提条件
- **OS**: Windows 10 または 11 (64bit)
- **Runtime**: .NET 10 デスクトップランタイム
- **GPU**: 推奨 NVIDIA GPU (VRAM 4GB以上)。CPUのみでも動作しますが、画像生成に数分を要する場合があります。

### 1.2 初期セットアップ (自動/手動)
1. **FFmpeg**: 初回起動時に自動ダウンロードを提案します。手動の場合は `ffmpeg.exe` をソフトと同じフォルダに置いてください。
2. **AIモデル**: `Models/` フォルダに適切なモデルを配置してください。
    - **LLM**: `.gguf` 形式。例: `phi-3-mini.gguf`
    - **画像生成**: `.safetensors` または `.bin` 形式。

## 2. 操作ガイド
### 2.1 台本生成の基本
1. **記事入力**: 右側の入力欄に、動画のソースとなるブログテキストを貼り付けます。
2. **設定調整**: 
    - **ターゲット時間**: ショート動画なら 15s 〜 60s を推奨。
    - **スタイル**: 「煽り系」「教育系」などを選択するとナレーションの口調が変化します。
3. **生成開始**: 「生成」ボタンを押すと、AIが素材を解析し、複数のバリエーションを提案します。

### 2.2 ブラッシュアップ機能 (Advanced)
生成されたカード内の「ブラッシュアップ」ボタンを押すと、AIがナレーションをよりプロフェッショナルな言い回しに、テロップをよりキャッチーな短いフレーズに書き換えます。

### 2.3 動画の書き出し (Rendering)
1. **Export**: 「動画を書き出し」をクリックします。
2. **進捗確認**: 画面下部のプログレスバーとステータステキストに現在の工程（音声生成中、レンダリング中など）が表示されます。
3. **完了**: 指定した保存フォルダに MP4 ファイルが出力されます。

## 3. パフォーマンスとトラブルシューティング
### 3.1 高速化のヒント (v1.8.2)
本バージョンは「並列パイプライン」を搭載しています。
- **GPU加速**: 設定で「GPU加速」を有効にすると、LLMの推論と画像生成が劇的に速くなります。
- **マルチタスク**: セマフォ制御により、画像生成中も CPU が空いていれば動画レンダリングを並列で行います。

### 3.2 ログの確認方法
不具合やエラーが発生した場合は、ソフトと同じフォルダにある `app_log.txt` を確認してください。
- **SD Generation failed**: プロンプトが複雑すぎるか、VRAM不足の可能性があります。
- **FFMpeg failed**: `ffmpeg.exe` が正しい場所にあり、壊れていないか確認してください。

### 3.3 よくあるエラー
- **モデルのロードに失敗する**: モデルファイルのパスに日本語（全角文字）が含まれているとエラーになる場合があります。
- **音声が出ない**: Windowsの「設定 > 時刻と言語 > 音声認識」で日本語の音声合成エンジンがインストールされているか確認してください。

## 4. 注意事項
- **オフライン動作**: 本ソフトは外部サーバーへデータを送信しません。
- **ライセンス**: 生成されたコンテンツの権利は、使用したAIモデルのライセンス（CreativeML Open RAIL-M等）に従います。
